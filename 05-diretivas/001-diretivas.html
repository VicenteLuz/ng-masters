ng-masters: AngularJS na prática

Angular pra gente grande = diretivas

Usando diretivas vc pode:

  • Criar tags customizadas, tipo:
  <div>
    <!-- componente que busca issues no Github -->
    <github-issue-list></github-issue-list>
  </div>
  • Modificar o comportamento de elementos existentes
  <!-- Uma diretiva "droppable" que transforma qualquer 
  elemento num alvo de drag and drop -->
  <div class="list" droppable ondrop="addItem(item)">
  </div>

Tô entendo aonde vc quer chegar...

Exatamente. Reuso! Crie seus componentes como diretivas, e saia reusando por aí.

É isso que faz do Angular uma parada tão foda.
