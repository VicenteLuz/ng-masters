jQuery = $

O jquery cria uma variável global chamada $.
Esse $ sabe fazer muita coisa. 
    	$ //$ eh uma função
    	$() //que retorna um array... (por quê? - Jajá a gente volta aqui)
    	
    	//$ tem atributos (que sao outras funcoes)

    	$.forEach //loop

Seleção e Manipulação

A função $ serve pra selecionar partes do DOM.
Uma vez selecionadas, elas podem ser manipuladas.

Filho Neto 
    	//selecionando via seletores CSS...
    	$('.vovo')
    	$('.titio')
    	$('#zequinha')

    	//Manipulando
    	$('.titio').text() //pega o conteudo como texto
    	$('.titio').text('Tioooo!') //seta o conteudo como texto
    	$('.titio').html() //pega o conteudo como html
    	$('.titio').html('olha o <strong>negrito</strong>!') //seta html
    	$('.titio').addClass('redback'); //adiciona classe css
    	$('.titio').css('width', '40px'); //modifica atributo css
    	$('.titio').attr('boi', 'lambeu'); //modifica atributo do elemento
    	$('.titio').removeClass('redback'); //remove classe css
