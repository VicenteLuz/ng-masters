Gulp - instalação

Node.js

No windows: baixa e instala - nodejs.org

No linux: baixa e instala

  sudo apt-get install nodejs npm

Testa: abre o dos (ou bash) e digita "node" (ou "nodejs")

Gulp

O node tem uma outra ferramenta associada, o npm, que é um gerenciador de pacotes. É ele que a gente vai usar pra instalar o grunt (e muitas outras coisas depois). É só abrir um terminal e mandar:

  sudo npm install -g gulp

Pronto. Isso instala o gulp no seu sistema operacional. Vc só precisa fazer isso uma vez na vida.

package.json

O package.json é um arquivo json que diz pro npm uma lista de coisas pra ele baixar

  cd gulp1
  npm install
  • Instala tudo no diretório atual
  • Olha, apareceu uma pasta "node_modules"
  • Tem rodar isso sempre que adicionar algo no package.json

gulpfile.js

O gulpfile.js é um arquivo javascript que diz pro gulp as tarefas que ele tem que executar.
